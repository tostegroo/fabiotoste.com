# fabiotoste.com
My beaultiful web site:

## Dependencies
grunt


## Setup
Install the grunt node modules

```
cd /src
```

And then...

```
npm install 
```

## Running

All dev files lives in **/src**

```
cd /src
```

Start grunt dev tasks via

```
grunt dev
```

## Deployment

To deploy all application to a single inline html file

```
cd /src
```
```
grunt prod
```
